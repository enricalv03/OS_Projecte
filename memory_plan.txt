================================================================================
BITMAP PLANNING
================================================================================

Total Pages to Track: 32,512 pages (from Entry 4)

Bitmap Size Calculation:
  - 1 bit per page
  - 32,512 pages / 8 bits per byte = 4,064 bytes
  - Round up to page boundary: 4,096 bytes = 1 page (4 KB)

Bitmap Location Options:
  Option 1: 0x00100000 (1 MB, first page of free memory - uses one free page)
  Option 2: 0x00101000 (1 MB + 4 KB, keeps first page free - RECOMMENDED)
  Option 3: Right after page tables (~0x00005C00) - NOT SAFE (in reserved area)

RECOMMENDED: 0x00101000 (1 MB + 4 KB)
  - Safe location (above all reserved areas, in Entry 4)
  - Keeps first page (0x00100000) free for allocation
  - Easy to remember
  - Won't conflict with anything
  - Reserve this 4 KB page in the bitmap!

Bitmap Memory Range: 0x00101000 to 0x00102000 (4 KB)

================================================================================
BITMAP INDEXING
================================================================================

Important: The bitmap tracks pages starting from Entry 4 (0x00100000)

Page Address to Bitmap Index:
  - Physical address: 0x00100000 = page index 0
  - Physical address: 0x00101000 = page index 1
  - Physical address: 0x00102000 = page index 2
  - etc.

Formula: bitmap_index = (physical_addr - 0x00100000) / 4096

Bit Position in Byte:
  - Byte index = bitmap_index / 8
  - Bit position = bitmap_index % 8
